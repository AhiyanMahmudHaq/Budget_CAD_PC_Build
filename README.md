# Budget_CAD_PC_Build

### Preface

This project is dedicated to open-sourcing the specifications of the Personal Computer (PC) that I use for Computer Assisted Design (CAD) of 3D models and Printed Circuit Boards (PCBs), so that other hobbyists can use this as a template for a budget-friendly CAD PC build. Please note that this was designed and planned in the year 2026, when a shortage of consumer NAND-based computer parts (such as Solid State Drives (SSDs) and Random Access Memory (RAM) modules) was happening. This is because of the influx of Artificial Intelligence (AI) companies demanding a massive abundance of these computer parts from manufacturers like Micron, SK Hynix, Samsung, etcetera.
